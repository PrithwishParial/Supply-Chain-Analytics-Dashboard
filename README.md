# 📊 Supply Chain Analysis Dashboard | Power BI Project

## Overview

This Power BI project focuses on analyzing supply chain operations to uncover insights related to procurement, logistics, supplier performance, and order fulfillment. The dashboard is designed to help businesses monitor operational efficiency, identify bottlenecks, and support data-driven decision-making.

---

## Key Objectives

* Analyze supplier performance and delivery reliability
* Monitor lead times and order fulfillment trends
* Evaluate procurement efficiency and operational KPIs
* Identify delays, risks, and supply chain bottlenecks
* Create an interactive dashboard for business stakeholders
